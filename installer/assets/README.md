# Assets para el Instalador

## icon.ico

Coloca aquí un archivo `icon.ico` para el instalador y la aplicación.

### Requisitos:
- Formato: ICO
- Tamaños recomendados: 16x16, 32x32, 48x48, 256x256
- Nombre del archivo: `icon.ico`

### Recursos para crear iconos:
- https://convertico.com/ - Convertir PNG a ICO
- https://realfavicongenerator.net/ - Generador de iconos
- https://www.flaticon.com/ - Iconos gratuitos

### Si no tienes icono:
Puedes comentar la línea `SetupIconFile` en `setup.iss` para usar el icono por defecto de Inno Setup.
